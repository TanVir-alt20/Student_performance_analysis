# Student Performance Analysis

## About This Project
Analysis of student exam performance data using R.
Dataset contains 1000 students with math, reading
and writing scores.

## What I Did
- Imported data using read.csv, read_csv and fread
- Cleaned and renamed columns
- Converted categorical variables to factors
- Created new columns: TotalScore, AvgScore, Grade, Result
- Wrote custom descriptive statistics function
- Performed EDA using dplyr, psych and skimr
- Stored data in SQLite database
- Exported results as CSV and Excel
- Used ggplot2 for visualization and RMarkdown for report

## Tools Used
- R
- dplyr
- data.table
- naniar
- skimr
- psych
- DBI / RSQLite
- writexl
- ggplot2, plotly & RMarkdown

## Key Findings
- Male students scored higher in math (68.7 vs 63.6)
- Female students scored higher in reading and writing
- 89.7% of students passed overall
- Test preparation course improved average scores
